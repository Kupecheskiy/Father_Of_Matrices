# Father_Of_Matrices

Бот для работы с матрицами в Telegram. Позволяет выполнять операции сложения, вычитания, умножения, транспонирования, вычисления определителя, обратной матрицы и ранга.

## Возможности

- Сложение, вычитание, умножение матриц
- Транспонирование
- Определитель (квадратные матрицы)
- Обратная матрица (квадратные невырожденные)
- Ранг матрицы
- Кнопка «Отмена» для прерывания операции
- Понятные сообщения об ошибках

## Используемые библиотеки

| Библиотека | Назначение |
|------------|------------|
| `aiogram` | Асинхронный фреймворк для Telegram Bot API |
| `numpy` | Матричные вычисления (сложение, умножение, определитель, ранг и др.) |
| `python-dotenv` | Загрузка токена из файла `.env` |
